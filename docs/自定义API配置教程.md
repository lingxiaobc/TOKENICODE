# 🔌 在 TOKENICODE 中接入自定义 API 提供商

> 本文面向完全不懂技术的用户。跟着截图一步步操作，10 分钟搞定。

---

## 前情提要：什么是「自定义 API」？

### AI 模型住在「服务器」上，你要通过「管道」去调用它

ChatGPT、Claude、DeepSeek……这些 AI 模型不住在你的电脑里，它们运行在各家公司的服务器上。你每次发消息，其实是通过网络把问题发过去，再把回答接回来。

这条「发出去 → 接回来」的通道，就叫 **API**。

### 那「API Key」是什么？

你把 API 想象成一扇门。
API Key 就是**开门的钥匙**——是服务商发给你的专属凭证，用来证明「这个请求是我的、费用算我的账」。

没有 Key，门不开。Key 错了，也进不去。

### 那「Base URL」又是什么？

门在哪栋楼？Base URL 就是**门的地址**。
你得告诉软件：去哪里找这扇门，再用 Key 开它。

### 为什么叫「自定义」？

市面上卖 AI API 访问权限的平台有很多，价格、速度、支持的模型各不相同。
TOKENICODE 不强制绑定某一家，你可以**自己选供应商、自己填地址和钥匙**——这就是「自定义 API」。

常见的供应商有：

- [云雾 AI](https://yunwu.ai)
- [硅基流动](https://siliconflow.cn)
- [DeepSeek](https://platform.deepseek.com)
- [月之暗面 Kimi](https://platform.moonshot.cn)
- [阿里云百炼](https://bailian.console.aliyun.com)
- ……以及更多

**本教程以「云雾 AI」为例**演示完整配置流程。
不同供应商的操作界面不同，但核心步骤完全一样：
**注册 → 充值 → 拿到 API Key + Base URL → 填入 TOKENICODE**

---

## 第一步：注册账号

打开浏览器，访问 [https://yunwu.ai](https://yunwu.ai)

[截图：云雾AI 首页]

点击右上角 **「注册」** 按钮，填写邮箱和密码完成注册。

[截图：注册页面]

> 💡 已有账号？直接登录，跳到第二步。

---

## 第二步：充值（先充点钱才能用）

登录后，点击顶部导航的 **「充值」** 或 **「余额」**。

[截图：充值入口]

按需选择金额，完成支付即可。

> 💡 新用户一般有免费额度，可以先试试，不够再充。

---

## 第三步：获取你的 API Key

在顶部导航找到 **「API 密钥」** 或 **「Key 管理」**，点进去。

[截图：API 密钥管理页面]

点击 **「创建新密钥」**，给它随便起个名字，比如 `TOKENICODE`。

[截图：创建密钥弹窗]

创建成功后，**复制那串以 `sk-` 开头的字符**，保存好，它只显示一次！

[截图：复制 API Key]

> ⚠️ 这个 Key 不要发给任何人，它相当于你账号的钥匙。

---

## 第四步：找到 API 接口地址（Base URL）

在供应商的文档或控制台里，找到它的 **API Base URL**，云雾 AI 一般是：

```
https://yunwu.ai/v1
```

[截图：文档/控制台里的 Base URL]

记下这个地址，下一步要用。

> 💡 不同供应商的地址不同，去对应平台的「文档」或「接入指南」里找，一般都会写清楚。

---

## 第五步：在 TOKENICODE 里添加这个 API

打开 TOKENICODE，点击左侧边栏的 **「设置」** 图标（⚙️）。

[截图：TOKENICODE 设置入口]

找到 **「API 提供商」** 或 **「自定义 API」** 选项，点击 **「添加」**。

[截图：添加 API 提供商页面]

按照下表填写信息：

| 字段 | 填什么 | 说明 |
|------|--------|------|
| 名称 | 随便起，比如「云雾AI」 | 只是给你自己看的标签 |
| Base URL | `https://yunwu.ai/v1` | 第四步找到的地址 |
| API Key | 刚才复制的 `sk-xxx...` | 第三步拿到的密钥 |
| 协议类型 | OpenAI 兼容 | 大多数供应商都选这个 |

[截图：填写 API 配置表单]

填完点 **「保存」** 或 **「测试连接」**。

---

## 第六步：测试是否连通

点击 **「测试连接」** 按钮。

[截图：测试连接按钮]

看到 **「连接成功」** 或 **「✅」** 提示，就大功告成了！

[截图：连接成功提示]

如果显示失败，最常见的原因：

| 报错 | 原因 | 解决 |
|------|------|------|
| 网络错误 | Base URL 填错，或平台服务异常 | 检查 URL 是否正确 |
| 认证失败 | API Key 不对或已失效 | 重新复制 Key，注意别带空格 |
| 余额不足 | 账号没钱了 | 去充值 |

---

## 完成！开始使用

回到 TOKENICODE 主界面，在模型选择下拉框里，选择刚才添加的提供商，就能用上对应的模型了。

[截图：选择云雾AI 作为提供商]
